This repository contains my solutions to **Array** problems from the **Apna College DSA Sheet** — one of the most popular and structured DSA curricula for C++ learners. Each solution is written in clean, well-commented **C++** and follows the logic taught in the course.

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/HariSingh6265/ApnaCollege_DSA_Arrays.git
   cd ApnaCollege_DSA_Arrays
   ```

2. **Compile any solution**
   ```bash
   g++ filename.cpp -o output
   ```

3. **Run the output**
   ```bash
   ./output
   ```

## 🧠 Why Arrays First?

Arrays are the most fundamental data structure in programming. Mastering arrays helps you:

- ✅ Understand memory allocation and indexing
- ✅ Solve problems using Sliding Window and Two Pointers
- ✅ Build the foundation for strings, matrices, and advanced DSA
